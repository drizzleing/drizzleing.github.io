# LessOrMore

关于
====================================
- 本站点的访问地址：<https://drizzleing.github.io>
- 上一个版本V1.0（存于jekyllbootstrap分支），是基于JekyllBootstrap搭建的博客站点，但JB模板较少，而且缺乏中文支持。
- 本次进行2.0版本的升级，一来改版样式更符合中国人的阅读习惯，二来对于整体结构和内容生产上做一些实践，尝试构建自己的知识库。
